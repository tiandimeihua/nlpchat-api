# nlpchat-api

---

## How to run

```bash
npm init
npm install underscore
npm install blueimp-md5
npm install axios
npm install qs
node post2.js
```

* [underscorejs](http://underscorejs.org/)
* [blueimp-md5](https://github.com/blueimp/JavaScript-MD5)
* [axios](https://github.com/axios/axios)
* [qs](https://www.npmjs.com/package/qs)
